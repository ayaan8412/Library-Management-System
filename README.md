# Library Management System

## Features
- **User Authentication**: Users can create accounts and log in securely.
- **Book Management**: Admins can add, remove, and update book information.
- **Search Functionality**: Users can search for books by title, author, genre, or ISBN.
- **Borrowing System**: Users can borrow and return books, keeping track of due dates.
- **Reservations**: Users can reserve books that are currently checked out.

## Usage
1. **Installation**: Clone the repository and install the dependencies.
   ```bash
   git clone https://github.com/ayaan8412/Library-Management-System.git
   cd Library-Management-System
   
   # Install dependencies (example for Node.js)
   npm install
   ```
2. **Running the Application**: Start the application using the following command:
   ```bash
   npm start
   ```
3. **Accessing the Application**: Open your web browser and go to `http://localhost:3000` to access the Library Management System.

## Functionality
- Users can register and log in to perform actions like searching and borrowing books.
- Admins have additional capabilities to manage the entire library's inventory.
- Notifications are sent for overdue books to maintain a smooth user experience.
